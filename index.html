X XML Repair Tool

Crossref XML Utility

DOI XML Repair

Metadata follows the new XML file. Article DOIs remain exactly from the old XML, including prefixes and suffixes. Output timestamp is incremented for the Crossref update.

Quick Start Guide & How to Use
  1. Old Data (Step 1): Upload your old Crossref XML file, or select Enter old DOIs to paste one DOI per line and click Fetch Full Crossref Metadata.
  2. New XML (Step 2): Upload your new Crossref XML file containing the updated metadata.
  3. Configure URL Output: Choose whether to use the article URLs defined in your new XML or override them manually by entering new website URLs (one URL per line).
  4. Verify DOI Mapping: The tool will automatically align articles. If the count differs or you make custom adjustments, correct the pairs using the dropdown selections.
  5. Generate XML: Click Regenerate if manual adjustments are made. Finally, click Copy or Download to export your repaired XML file.
